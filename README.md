# michael.recachinas.dev

Personal website built with Jekyll. Minimalist profile page with dark/light mode toggle.

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Visit http://localhost:4000

## Configuration

Edit `index.markdown` frontmatter to update:
- Name and title
- Social links (GitHub, LinkedIn, Twitter, Mastodon, email)

Profile image is `assets/images/profile.jpg`.

Edit `about.markdown` frontmatter to update work timeline.
